# Banner Images

Place your hero banner image here with the name: `hero-banner.jpg`

**Recommended image specifications:**

- **Name**: `hero-banner.jpg`
- **Dimensions**: 1920x600px or 1920x800px (wider for desktop, maintains aspect ratio)
- **Format**: JPG or PNG
- **Size**: Max 500KB for optimal loading time
- **Content**: Professional background image for the home page hero section

The image will be displayed behind the text:

- Premium Quality
- JOURNALS & BOOKMARKS
- Discover our exquisite collection of premium journals and bookmarks
- Shop Now button
